# JSP/Servlet

## 주제
Swing 기반 미용실 관리 프로그램 -> JSP/Servlet
<br><br>

## 사용언어 및 기술 
- Server : Apache Tomcat
- Database : Oracle 11g
- Language : JAVA, JSP, HTML/CSS, JS(jQuery)
